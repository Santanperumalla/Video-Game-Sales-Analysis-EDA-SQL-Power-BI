# 🎮 Video Game Sales Analysis

## 📌 Project Overview

This project performs an **Exploratory Data Analysis (EDA)** on a Video Game Sales dataset to uncover trends, patterns, and insights related to game sales across different regions, platforms, genres, and publishers.

The analysis helps answer important business questions such as:

* Which video games generated the highest global sales?
* Which gaming platforms are the most successful?
* Which genres dominate the gaming industry?
* How do sales vary across different regions?
* Which publishers contribute the most to overall sales?

---

## 📂 Dataset Information

The dataset contains information about video game sales from various gaming platforms around the world.

### Features

| Column       | Description                       |
| ------------ | --------------------------------- |
| Rank         | Ranking based on global sales     |
| Name         | Name of the video game            |
| Platform     | Gaming platform                   |
| Year         | Release year                      |
| Genre        | Genre of the game                 |
| Publisher    | Publishing company                |
| NA_Sales     | Sales in North America (Millions) |
| EU_Sales     | Sales in Europe (Millions)        |
| JP_Sales     | Sales in Japan (Millions)         |
| Other_Sales  | Sales in Other Regions (Millions) |
| Global_Sales | Total Worldwide Sales (Millions)  |

### Dataset Size

* Rows: 16,598
* Columns: 11

## 🔍 Key Insights

### Genre Analysis

* Action is the most popular genre with over 3,300 games.
* Sports is the second most common genre.
* Puzzle and Strategy genres have the fewest titles.

### Platform Analysis

* A small number of platforms contribute the majority of global sales.
* Popular console generations dominate the sales charts.

### Sales Trends

* Certain years experienced significant growth in game sales.
* Blockbuster titles create noticeable spikes in global revenue.

### Regional Analysis

* North America contributes a major share of global sales.
* A strong positive relationship exists between North American sales and worldwide sales.
* High-selling games in North America often become global bestsellers.
